# HWP → HWPX 일괄 변환기

한글(HWP) 파일을 한글 2014 이상 형식인 HWPX 파일로 일괄 변환하는 프로그램입니다.

## ✨ 주요 기능

- 📁 입력/출력 폴더 지정
- 🔄 HWP → HWPX 일괄 변환
- 📂 하위 폴더 포함 검색 옵션
- 📊 실시간 진행률 표시
- 📝 변환 로그 기록
- 🖥️ 직관적인 GUI 인터페이스

## 🖼️ 프로그램 화면

프로그램 실행 시 간단한 GUI 화면이 나타납니다:
- 입력 폴더: 변환할 HWP 파일들이 있는 폴더
- 출력 폴더: 변환된 HWPX 파일을 저장할 폴더
- 하위 폴더 포함 옵션
- 변환 진행률 표시
- 실시간 로그

## 📋 사용 방법

### 1️⃣ 실행하기
- `HWP_to_HWPX_Converter.exe` 파일을 더블클릭하여 실행

### 2️⃣ 폴더 선택
- **입력 폴더**: 변환할 HWP 파일들이 있는 폴더 선택
- **출력 폴더**: 변환된 HWPX 파일을 저장할 폴더 선택

### 3️⃣ 옵션 설정
- **하위 폴더 포함**: 체크 시 입력 폴더의 하위 폴더까지 검색

### 4️⃣ 변환 시작
- **변환 시작** 버튼 클릭
- 진행률과 로그를 확인하며 완료 대기

### 5️⃣ 완료
- 변환 완료 메시지 확인
- 출력 폴더에서 변환된 HWPX 파일 확인

## ⚠️ 필수 요구사항

### 시스템 요구사항
- **운영체제**: Windows 7 이상
- **한글 프로그램**: 한글 2007 이상 (반드시 설치 필요)
- **메모리**: 2GB 이상 권장

### 중요 사항
1. **한글 프로그램이 설치되어 있어야 합니다**
   - 프로그램이 한글을 자동으로 제어하여 변환 작업 수행
   
2. **변환 중에는 한글이 백그라운드에서 실행됩니다**
   - 작업 중 한글 프로그램을 직접 종료하지 마세요
   
3. **대용량 파일 변환 시 시간이 소요됩니다**
   - 파일 크기와 개수에 따라 시간이 다를 수 있습니다

## 🔧 개발자 정보

### 개발 환경 구성 (개발자용)

#### 1. Python 설치
- Python 3.8 이상 필요

#### 2. 패키지 설치
```bash
pip install -r requirements.txt
```

#### 3. 프로그램 실행
```bash
cd src
python main.py
```

#### 4. EXE 파일 빌드
```bash
build.bat
```
또는
```bash
cd src
pyinstaller --onefile --windowed --icon=../icon.ico --name="HWP_to_HWPX_Converter" main.py
```

빌드된 실행 파일은 `src/dist/` 폴더에 생성됩니다.

## 🐛 문제 해결

### 한글 초기화 실패
- 한글 프로그램이 설치되어 있는지 확인
- 한글 프로그램을 한 번 실행한 후 정상 종료
- 관리자 권한으로 프로그램 실행

### 변환 실패
- 입력 파일이 손상되지 않았는지 확인
- 한글에서 직접 파일을 열어서 확인
- 출력 폴더에 쓰기 권한이 있는지 확인

### 프로그램이 느림
- 대용량 파일이 많을 경우 시간이 소요될 수 있음
- 한글이 백그라운드에서 작동하므로 다른 작업을 진행해도 무방
